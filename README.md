# VocaFlow

好吧, 实际上它在vs里面的项目名字叫做VocaFlow. 因为我的初衷是做一个AI生成歌词罗马音的应用(QQ音乐不支持复制歌词, 而且很多歌也没有罗马音).

但写着写着, 我想, 加一个记录生词的功能是不是夯爆了? 然后就加了, 还加入了英语的支持, 还写了一个背词的功能, 再之后我想, 做都做了, 那就让它能记录任意文本的生词吧.

于是它比起只是一个歌词转罗马音的工具, 似乎多了点什么东西, 以至于那个名字不太能完全描述它了.

然后在听从Gemini谏言之后, 我决定给她另一个夯爆的名字: VocaFlow.

VocaFlow, 意思是"词汇的流动", 希望它能帮助你更好地学习和掌握新词汇, 让词汇在你的脑海中流动自如.

本app支持日语/英语, 可以输入任意文本要求AI找出生词并记录到记忆中, 之后可以查阅也可以背诵.

除此之外, 支持搜索QQ音乐的歌词并转换成罗马音输出. 如果觉得默认prompt不满意, 也可以自行修改.

使用本app需要你有OpenRouter的API Key, 申请简单, 支持国内支付. 具体办法请自行搜索.

如果觉得本app对你有帮助, 欢迎给个star⭐️, 也欢迎提issue或者PR, 我会尽量抽时间改进它.
